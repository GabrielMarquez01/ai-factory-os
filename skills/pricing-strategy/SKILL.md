# 💲 Pricing Strategy

> **Analogía:** El precio no es solo un número — es un mensaje. Un precio de $99 dice algo diferente a $100, aunque la diferencia sea $1. Un precio de $49/mes dice algo diferente a $499/año, aunque el ingreso anual sea casi el mismo.
>
> El precio incorrecto puede matar un producto bueno. El correcto puede salvar uno regular.

## El Error Más Común: Precio Basado en Costo

```
Error: Precio = Costo × (1 + Margen deseado)
              = $50 costo × 2 = $100

Problema: el mercado no sabe ni le importa cuánto te costó producir algo.
El mercado paga por el VALOR PERCIBIDO, no por tu estructura de costos.
```

**La pregunta correcta no es "¿cuánto me costó?" sino "¿cuánto vale para el cliente?"**

## Las 3 Estrategias de Precio (y Cuándo Usar Cada Una)

### Estrategia 1: Precio de Penetración

Entras con un precio bajo para ganar usuarios rápido. Subes después.

```
Cuándo usar: mercado establecido con competidores más caros,
             necesitas masa crítica antes de ganar dinero

Riesgo: atraes usuarios sensibles al precio (que se irán cuando subas)
        y señalas "esto es barato" (afecta la percepción de calidad)

Ejemplo: Spotify vs el precio de los CDs
```

### Estrategia 2: Precio Premium / Valor Percibido

Precio alto que comunica calidad y selecciona al cliente correcto.

```
Cuándo usar: producto diferenciado, nicho de alto valor,
             prefieres pocos clientes que pagan bien vs muchos que pagan poco

Ventaja: menos clientes para atender = mejor servicio = mayor retención
         el cliente premium suele tener menos churn

Ejemplo: Notion es "caro" vs Evernote, pero tiene usuarios más fieles
```

### Estrategia 3: Precio de Competencia + Diferenciación

Precio similar al competidor pero con propuesta de valor diferente.

```
Cuándo usar: mercado precio-sensitivo con competidores establecidos
             cuando tu diferenciación no es el precio sino el servicio

Táctica: misma cancha de precio, ganas en otro atributo
         (mejor soporte, idioma local, integración con herramientas que usan)
```

## Precio Para LatAm: La Ecuación Diferente

El poder adquisitivo en México, Colombia, Argentina, etc. es diferente al de EE.UU. o Europa.

**Referencia de conversión de precio MX vs US:**

| Precio US | Precio MX razonable | Lógica |
|-----------|--------------------|----|
| $10/mes   | $99-149 MXN/mes   | 40-60% del equivalente en dólares |
| $29/mes   | $249-349 MXN/mes  | Ajuste de PPP (paridad de poder de compra) |
| $99/mes   | $699-899 MXN/mes  | Precio de entrada profesional MX |
| $299/año  | $1,999-2,999 MXN  | Pago anual con descuento implícito |

**Regla:** investiga cuánto cobran los líderes del mercado en tu nicho en MX específicamente, no solo en USD.

## El Modelo de Precios Para SaaS/Apps: Good-Better-Best

El modelo de 3 planes no es capricho — es psicología de la decisión:

```
Plan Básico  → ancla de precio baja (hace que el medio parezca razonable)
Plan Pro     ← el que quieres que compren (diseñado para ser la opción obvia)
Plan Business → ancla de precio alta (hace que el Pro parezca una ganga)
```

**Cómo diseñar el plan Pro para que sea la opción obvia:**
- Incluye TODO lo del plan Básico + las funciones más deseadas
- El precio es 2-3x el Básico pero contiene 5x el valor percibido
- El plan Business existe principalmente para hacer que el Pro "parezca razonable"

### Ejemplo de Estructura

```
BÁSICO — $99 MXN/mes
  ✓ Función A
  ✓ Función B
  ✓ Hasta 10 [unidad]

PRO — $249 MXN/mes   ← el que quieres vender
  ✓ Todo lo del Básico
  ✓ Función C (la más deseada)
  ✓ Función D
  ✓ Hasta 100 [unidad]
  ✓ Soporte prioritario
  🌟 "MÁS POPULAR"

BUSINESS — $799 MXN/mes
  ✓ Todo lo del Pro
  ✓ [Función enterprise]
  ✓ Ilimitado
  ✓ Onboarding dedicado
```

## Pricing Anual vs Mensual

El plan anual es el mejor mecanismo anti-churn disponible:

```
Mensual: $249 MXN/mes
Anual:   $1,999 MXN/año (≈ $167/mes → 33% de descuento)

Para el cliente: paga menos por mes
Para ti: dinero adelantado + churn = 0 por 12 meses
```

**Cómo promover el plan anual:**
- Muestra el precio mensual equivalente: "$167/mes — pagado anualmente"
- Calcula el ahorro en pesos reales: "Ahorras $990 al año"
- Badge visual: "Ahorra 33%" con color que resalte

## Cuándo y Cómo Subir el Precio

Subir el precio es la acción más resistida por los fundadores y la más rentable.

**Señales de que tu precio es muy bajo:**
- Los clientes aceptan sin negociar (sin rozamiento)
- Tienes más demanda de la que puedes atender
- Tus clientes dicen que "es barato para lo que ofrece"
- Tu margen no permite reinvertir en el producto

**Cómo subir sin perder clientes:**

```
Opción A: Grandfathering (la más amigable)
→ Los clientes actuales mantienen su precio para siempre
→ Nuevos clientes pagan el precio nuevo
→ Comunicar con 30-60 días de anticipación

Opción B: Subida con previo aviso
→ "A partir del [fecha], el precio sube a $X"
→ "Renueva ahora y queda fijo por 12 meses al precio actual"
→ Crea urgencia real para quienes estaban indecisos

Opción C: Por tiers de uso
→ El precio base no cambia
→ Nuevas funcionalidades se agregan al tier superior
→ Migración natural, no forzada
```

## Experimentos de Precio

El precio correcto no se encuentra en un libro — se descubre con experimentos:

```
Experimento básico:
1. Define dos precios a probar (ej: $149 vs $249)
2. Muestra cada precio a un grupo diferente de visitantes
3. Mide la tasa de conversión de cada grupo
4. El precio que maximiza INGRESO (no solo conversiones) gana

Ingresos = Tasa de conversión × Precio × Volumen de tráfico

Ejemplo:
Precio $149: 5% conversión → $149 × 50 clientes = $7,450
Precio $249: 3% conversión → $249 × 30 clientes = $7,470

→ $249 gana aunque convierte menos — ganas más dinero con menos clientes
```

## La Regla del Precio

> El precio que sientes que "quizá es mucho" probablemente es el correcto.
>
> Si nunca nadie te dice "esto está caro" → estás cobrando poco.
> Si todos dicen "está caro" → quizá cobrando mucho, o explicando mal el valor.
> El punto óptimo: algunos dicen "está caro" pero la mayoría lo considera justo.
