# Email Sequence AI Skill

## Propósito
El email es el canal con mayor ROI ($42 por cada $1 invertido). Este skill genera
secuencias completas, listas para importar en Kit (ConvertKit), Mailchimp, Klaviyo
o cualquier herramienta de email marketing.

---

## Activadores
- Usuario necesita automatizar la comunicación post-lead magnet
- Usuario quiere aumentar conversión de su funnel
- Parte del flujo de `sales-funnel-builder`
- Usuario quiere reducir churn o reactivar inactivos

## Reglas de Copywriting para LATAM

**Subject lines que funcionan:**
- Pregunta directa: "¿Por qué [problema]?"
- Personalización: "[Nombre], tienes [X] días para..."
- Curiosidad: "El error que comete el 87% de [audiencia]"
- Urgencia real: "Cierra hoy a las 11:59pm"
- Número específico: "3 pasos para [resultado] esta semana"
- Emoji estratégico: solo 1 al inicio o final, nunca en medio

**Estructura de email que convierte:**
1. **Hook (primera línea):** debe capturar atención en preview
2. **Contexto (2-3 líneas):** por qué les escribes hoy
3. **Valor (bullet points):** qué obtienen, no qué es
4. **CTA único (1 botón):** un email = una acción
5. **P.D.:** siempre incluir — es lo segundo más leído después del asunto

**Palabras que evitar (activan spam):**
gratis, GRATIS, Gana dinero, ingresos pasivos, oferta especial URGENTE,
cien por ciento, garantía, haz clic aquí, compra ya

## Configuración técnica

### Para Kit (ConvertKit):
```
Secuencia → New Sequence → Add Email (repite por cada email)
Configurar: Day X, Trigger: Previous email opened/sent
Agregar tags automáticas: "comprador-[producto]", "lead-magnet-[nombre]"
```

### Para Mailchimp:
```
Automations → Customer Journeys
Starting Point: Suscripción a lista / Tag agregado
Journey → Add Email → Set delay → Add Email...
```

### Para n8n (automatización completa):
Ver `n8n-workflow-generator` → Secuencia LEAD MAGNET → NURTURE AUTOMÁTICO
